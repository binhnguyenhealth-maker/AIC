# Showcase assets

These images are real iPhone Simulator captures from the debug build. The home
screen uses the app's debug-only `--aic-showcase=home` launch argument so the
portfolio state is reproducible without a live account or private credentials.
Release builds do not compile the showcase fixture.

The landscape `social-preview.png` is sized at 1280 x 640 for GitHub's repository
social preview. Upload it in **Settings → General → Social preview** after changes
to the image have been reviewed.
